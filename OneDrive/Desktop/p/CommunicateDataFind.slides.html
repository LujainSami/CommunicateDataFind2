Communicate Data Findings Project¶

The project have two parts which demonstrate the value of data visualization techniques in the data analysis process and why is it important. First part of the project, I will be using the Python visualization libraries to systematically explore the dataset that was selected, started from the plots of each single variables and build up plots of multiple variables.The second part, I will illustrates the interesting properties, trends, and the relationships that was discovered in the selected dataset. The primary method of conveying the findings through the transforming of the exploratory visualizations from the first part into polished, explanatory visualizations.
